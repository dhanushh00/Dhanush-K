## 🛠 Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Theming:** Next-Themes (Dark/Light mode support)

## ✨ Features

- **Modern UI/UX:** Clean, uncluttered interface designed with a focus on typography and spacing.
- **Responsive Design:** Fully responsive layout that looks great on mobile, tablet, and desktop devices.
- **Scrollspy Navigation:** Sidebar navigation automatically highlights the active section as you scroll.
- **Dark/Light Mode:** Seamless theme switching with optimized color palettes for both modes.
- **Interactive Elements:** Smooth hover animations on project cards, buttons, and links using Framer Motion.
- **Performance Optimized:** Built with Next.js App Router for fast loading times and optimal SEO.

## 💻 Running Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/dhanushh00/Dhanush-K.git
   cd Dhanush-K
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

4. **Open in browser:**
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
